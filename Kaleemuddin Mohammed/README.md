# E-Commerce Checkout Flow Testing & Defect Report

## Objective
Perform end-to-end testing for an e-commerce checkout flow covering product selection, cart, checkout validation, coupon simulation, payment flow simulation, and order confirmation.

## Included Deliverables
- Test case document
- Test execution report
- Bug/defect report
- Screenshots of issues
- Final QA summary report
- README file

## Review Scope
- Product listing page
- Product details page
- Cart page
- Checkout form
- Coupon/discount validation
- Payment simulation
- Order confirmation page
- Desktop and mobile responsiveness

## Suggested Review Steps
1. Open the test case workbook.
2. Compare execution status against the test execution report.
3. Review defects by severity and priority.
4. Check evidence screenshots.
5. Read the final QA summary report.
